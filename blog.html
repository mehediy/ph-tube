Discuss the scope of var, let, and const

var has function scope, is hoisted, and lacks block scope, making it less predictable and generally discouraged in modern JavaScript. let and const both have block scope, but const is used for variables that can not be reassigned after initialization, while let is used for variables that may be reassigned.

What do you mean by REST API?

REST API, which stands for Representational State Transfer Application Programming Interface, is a set of architectural principles and constraints for designing and building web services. REST APIs use standard HTTP methods (GET, POST, PUT, DELETE, etc.) to perform CRUD (Create, Read, Update, Delete) operations on resources.
